Table

HTML tables allow web developers to arrange data into rows and columns <table> </table>
  • The <th> </th> tag defines a header cell in an HTML table.
  • The <tr> </tr> tag defines a row in an HTML table.
  • The <td> </td> tag defines a standard data cell in an HTML table.
